# Coffee Sales Analysis Dashboard

This project analyzes a coffee sales dataset using Microsoft Excel to find sales trends, customer insights, and product performance.

The workbook includes a complete dashboard with charts, pivot tables, and interactive filters built from 1000 sales records.

# Features

* Sales Dashboard
* Country-wise Sales Analysis
* Top 5 Customers Analysis
* Coffee Type Sales Comparison
* Pivot Tables & Charts
* Interactive Slicers
  
# Dataset Includes

* Orders Data
* Customers Data
* Products Data

# Key Findings

* Arabica, Excelsa, Liberica, and Robusta coffee sales were compared
* Some customers contributed significantly more revenue than others
* Sales performance varied across different countries
* Dashboard visualizations made trends easier to identify quickly
  
 # Tools Used

* Microsoft Excel
* Pivot Tables
* Pivot Charts
* Data Cleaning
* Dashboard Design

# Purpose

This project was created to practice real-world Excel data analysis and dashboard-building skills using a business sales dataset.
